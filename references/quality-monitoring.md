> 版本：1.0.0
> 更新时间：2026-06-12
> 目标：质量问题提前发现，数据驱动创作

---

## 概述

实时监控写作质量，包括一致性校验、爽点密度、AI腔趋势，帮助作者及时发现问题。

---

## 监控维度

| 维度 | 监控内容 | 触发时机 | 处理方式 |
|------|---------|---------|---------|
| 一致性校验 | 物品/穿衣/环境/时间线矛盾 | 每章写完后 | 自动校验，warning不阻塞 |
| 爽点密度 | 爽点间隔是否过大 | 每批写完后 | 超5000字无爽点标记warning |
| AI腔趋势 | 禁用词命中数变化 | 每批写完后 | 连续3批上升自动提醒 |
| 伏笔健康 | 伏笔埋设到回收的时间 | 每50章 | 超期伏笔预警 |

## 自动化流程

### 1. 写入时一致性校验

每章写完后，自动执行：

```python
def consistency_check(chapter_file, tracking_files):
    warnings = []
    
    # 检查物品位置
    items_in_text = extract_items(chapter_file)
    items_in_tracking = load_tracking("物品.md")
    for item in items_in_text:
        if item not in items_in_tracking:
            warnings.append(f"物品 {item} 未在追踪文件中记录")
    
    # 检查穿衣连续性
    clothing_in_text = extract_clothing(chapter_file)
    environment = load_tracking("环境.md")
    if "单衣" in clothing_in_text and environment["season"] == "冬":
        warnings.append("冬季穿单衣，不符合季节")
    
    # 检查时间线连续性
    timeline = load_tracking("时间线.md")
    chapter_time = extract_time(chapter_file)
    if chapter_time - timeline["last_time"] > timedelta(days=3):
        warnings.append(f"时间跳跃{ (chapter_time - timeline['last_time']).days }天，需确认")
    
    return warnings
```

### 2. 爽点密度监控

每批写完后，自动计算：

```python
def check_hook_density(chapters):
    issues = []
    
    for chapter in chapters:
        # 提取爽点位置
        hooks = extract_hooks(chapter)
        if not hooks:
            issues.append(f"{chapter.name}: 无爽点")
            continue
        
        # 计算爽点间隔
        last_hook_position = 0
        for hook in hooks:
            interval = hook.position - last_hook_position
            if interval > 5000:
                issues.append(f"{chapter.name}: 爽点间隔{interval}字，超过5000字阈值")
            last_hook_position = hook.position
    
    return issues
```

### 3. AI腔趋势追踪

每批写完后，自动统计：

```python
def track_ai_pattern(chapters):
    banned_words_count = 0
    ai_patterns_count = 0
    
    banned_words = load_banned_words()
    ai_patterns = ["仿佛", "犹如", "不禁", "心中涌起", "眼中闪过"]
    
    for chapter in chapters:
        text = chapter.read()
        for word in banned_words:
            banned_words_count += text.count(word)
        for pattern in ai_patterns:
            ai_patterns_count += text.count(pattern)
    
    return {
        "banned_words": banned_words_count,
        "ai_patterns": ai_patterns_count
    }
```
