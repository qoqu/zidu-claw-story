# 系统流题材模板

> **核心卖点**：主角脑中突然多了个"发布任务给奖励"的金手指，爽点是清晰的进度条式成长与"系统逼我变强"的喜剧张力。

## 1. 核心流派/细分
- **签到/打卡系统**：每日任务换资源，安逸养成。
- **成就/任务系统**：完成指定行为解锁能力，目标驱动。
- **反套路系统**：系统本身有毒/坑主，喜剧反差。
- **养成系系统**：种田/经营类系统，慢热满足。

## 2. 世界观/设定要素
- **系统规则**：任务、奖励、惩罚清晰量化，读者有预期。
- **面板数值**：等级、技能、积分，可视化成长。
- **系统人格**：毒舌/傲娇/坑货，增互动趣味。
- **限制条件**：冷却、失败惩罚，避免无敌无聊。

## 3. 经典爽点套路
- 任务打卡：顺手完成领神装。
- 系统坑主：发布社死任务被迫执行。
- 数值碾压：等级差让对手怀疑人生。
- 隐藏成就：意外触发超稀有奖励。
- 扮猪吃虎：系统加持却装弱。

## 4. 大纲节奏建议
- **开篇钩子**：以"醒来脑中多了一道机械音"立金手指。
- **中期结构**：任务难度阶梯上升，每段解锁新功能保持新鲜。
- **高潮设计**：系统终极任务/真相揭晓，主角完成跃迁或反控系统。

## 5. 常见雷区/禁忌
- 系统变自动托管，主角无主动性。
- 任务设计无聊（签到三千章）。
- 数值膨胀失控，后期无力。
- 系统话痨打断叙事节奏。
- 金手指太强失去冲突张力。
