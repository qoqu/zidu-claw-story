# 克苏鲁题材模板

> **核心卖点**：在不可名状的恐怖与理性崩溃边缘，人类渺小无力却仍要挣扎的"未知恐惧+理智对抗"快感。

## 1. 核心流派/细分
- **调查员流**：普通人接触异常事件，逐步拼凑真相，爽点是"在疯狂边缘保持清醒"。
- **旧日降临流**： cult 与神祇博弈，世界观宏大，适合写文明级灾难。
- **SCP/收容物风**：单篇式诡异实体，重设定猎奇，适合短中篇。
- **深海/宇宙恐怖**：以"广阔未知"压迫感为主，弱化战斗、强化心理。

## 2. 世界观/设定要素
- **不可知神祇**：旧日支配者、外神，力量维度远超人类，禁止正面战胜，只能拖延或封印。
- **理智值(SAN)系统**：接触异常会掉 sanity，可作为数值化紧张感来源。
- **诡异法则**：如"不能数清台阶""凝视雕像会梦游"，规则怪谈式禁忌增强代入。
- **秘密结社**：深潜者、星之眷族等 cult 组织，提供人间反派。

## 3. 经典爽点套路
- 线索拼图：零散日记、符号逐步指向一个不敢说的真相。
- 反转认知：以为的救助者是祭品操盘者。
- 极限逃生：理智将崩时赌一把脱离不可名状之握。
- 以凡抗神：用谎言/仪式/牺牲暂时封印而非战胜。
- 同类相残：被污染者不再是人，队友变威胁。

## 4. 大纲节奏建议
- **开篇钩子**：一封奇怪遗书/小镇失踪案/一个不会死的尸体，立刻制造"不对劲"。
- **中期结构**：调查层层深入，每揭露一层真相就放大恐惧尺度，sanity 持续承压。
- **高潮设计**：直面实体时主角不"打赢"而是"活下来并带出秘密"，留开放式绝望。

## 5. 常见雷区/禁忌
- 把克苏鲁写成"能练级打败的怪"，彻底失去内核。
- 恐怖描写堆砌名词（"不可名状"滥用）却无具体画面。
- 主角降智送人头，失去读者共情。
- 收尾强行圆满治愈，消解虚无基调。
- 过度血腥直写易触审核，用心理暗示更克制有效。
