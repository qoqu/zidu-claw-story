# 民国言情题材模板

> **核心卖点**：乱世烽烟里"旗袍与枪"的浪漫，爽点是新旧交替时代女性挣脱礼教、与锋利男主谈一场危险又炽热的爱。

## 1. 核心流派/细分
- **谍战恋**：特工夫妇假戏真做，家国与爱两难。
- **军阀少帅**：枭雄×名媛，强强拉扯。
- **梨园/才女**：戏子或新女性被权贵倾心的虐甜。
- **留学新女性**：从旧家庭出走，独立与爱情并行。

## 2. 世界观/设定要素
- **时代切片**：军阀混战、北伐、租界、报刊业，给背景质感。
- **新旧冲突**：缠足与放足、包办与自由婚恋的张力。
- **阶层与势力**：军阀、买办、革命党、青帮，多方博弈舞台。
- **审美符号**：旗袍、百乐门、电报、黄包车，画面感来源。

## 3. 经典爽点套路
- 冲破包办：退掉旧婚约追自由爱。
- 并肩作战：乱世里互为后方。
- 身份反转：看似弱女子是地下者。
- 少帅低头：枭雄为她收锋芒。
- 生死诀别：时代碾压下的最美告白。

## 4. 大纲节奏建议
- **开篇钩子**：以"一纸婚约/一次暗杀擦肩"立乱世与缘分。
- **中期结构**：感情在时代事件（罢工、战事）中淬炼，甜虐交替。
- **高潮设计**：家国大义与个人爱欲的终极抉择，完成人物升华。

## 5. 常见雷区/禁忌
- 历史立场模糊、美化侵略势力红线。
- 男主大男子主义伪装深情惹反感。
- 时代细节错（称谓、物价）被挑。
- 把女性解放写浅、只剩恋爱。
- 强行圆满无视时代悲情底色。
