# 无限流题材模板

> **核心卖点**：被扔进一个个电影/游戏/副本世界求生的刺激，爽点是"凡人靠智勇与成长在不断死亡威胁中变强"。

## 1. 核心流派/细分
- **恐怖副本流**：进鬼片/规则副本求生，重氛围与解谜。
- **游戏化系统流**：属性面板、任务、奖励点，RPG 养成感。
- **团队博弈流**：队员各有心思，内斗+外敌双线。
- **单人破局流**：孤狼型主角靠脑子通关，硬核爽。

## 2. 世界观/设定要素
- **主神空间**：发布任务、发放奖励的中枢，规则需清晰。
- **副本生态**：每个世界独立法则（恐怖/武侠/末世），切换保持新鲜。
- **属性与技能**：力量、智力、技能树，量化成长。
- **死亡威胁**：失败即死的硬代价，张力来源。

## 3. 经典爽点套路
- 提前预知：看过原作的降维通关。
- 极限破规：用副本漏洞反杀 boss。
- 队伍反水：识破叛徒先下手。
- 技能 combo：组合技能秀操作。
- 主神真相：后期揭空间本质的大反转。

## 4. 大纲节奏建议
- **开篇钩子**：主角猝死/被选中，直接进入第一个新手副本立生死压。
- **中期结构**：副本难度螺旋上升，每个副本解锁新能力并埋主神暗线。
- **高潮设计**：终极副本或对抗主神，集合全书成长完成跃迁。

## 5. 常见雷区/禁忌
- 副本规则不清，通关靠作者金手指。
- 队员脸谱化、送人头惹烦。
- 战力膨胀失控，后期无力。
- 抄热门 IP 副本过多引版权/融梗争议。
- 死亡无重量，反复复活疲劳。
