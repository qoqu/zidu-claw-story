# 知乎短篇题材模板

> **核心卖点**："谢邀，人在现场，刚下飞机"式的第一人称亲历感，爽点是高密度反转+神结尾，三分钟读完一个完整爽故事。

## 1. 核心流派/细分
- **悬疑反转体**：前半铺垫、结尾一击反转型。
- **温情治愈体**：真实善意小故事，戳泪点。
- **职场/社会洞察体**：借故事讲道理，共鸣强。
- **脑洞沙雕体**：离谱设定+一本正经叙述的喜剧。

## 2. 世界观/设定要素
- **第一人称亲历**："我"的视角+细节真实感是灵魂。
- **回答体结构**：开头立题（如"有哪些细思极恐的瞬间"），正文作答。
- **节奏铁律**：短（1000-3000字）、起承转合紧凑、结尾需爆。
- **真实细节**：时间、地点、对话要像真事。

## 3. 经典爽点套路
- 神结尾反转：最后一句颠覆全文认知。
- 细思极恐：平淡叙述里藏一个恐怖细节。
- 爽文打脸：匿名爆料让恶人社死。
- 泪点收束：一个动作/一句话破防。
- 沙雕收尾：严肃铺垫落在一个离谱真相。

## 4. 大纲节奏建议
- **开篇钩子**：第一句直接进事件（"那天我差点死在电梯里"）。
- **中期结构**：只写必要信息，每个段落推进或设疑，不废话。
- **高潮设计**：结尾段给反转/升华，留白比解释更有力。

## 5. 常见雷区/禁忌
- 头重脚轻，铺垫长结尾弱。
- 反转靠巧合硬拗，逻辑不通。
- 写成说教长文失去短篇味。
- 编造灾难/事件引不适或违规。
- 标题党内容不符，读者觉得被骗。
