# 游戏体育题材模板

> **核心卖点**：从替补/菜鸟到冠军的汗水逆袭，爽点是"每一次训练、每一分比分都真实攒出来的荣耀"，热血代偿极强。

## 1. 核心流派/细分
- **电竞（并入电竞题材另列）此处重传统体育**：篮球/足球/田径的逆袭。
- **网游竞技**：副本首杀、公会战，虚拟荣耀。
- **全民健身/减肥逆袭**：素人变达人，治愈向。
- **教练/经理流**：幕后布局打造冠军团队。

## 2. 世界观/设定要素
- **项目规则**：真实赛事规则、战术体系，专业感来源。
- **训练数值**：体能、技术、默契，量化成长曲线。
- **对手梯度**：从校队到职业的阶梯，提供目标。
- **赛事日历**：真实/拟真联赛节点，节奏锚。

## 3. 经典爽点套路
- 绝杀翻盘：最后时刻逆天改命。
- 新秀爆发：默默无闻一战成名。
- 战术碾压：用脑子破强敌身体优势。
- 师徒情/队友情：更衣室羁绊。
- 复仇宿敌：当年输的人今朝赢回来。

## 4. 大纲节奏建议
- **开篇钩子**：以"被淘汰/坐板凳/遭嘲笑"立低谷与野心。
- **中期结构**：训练→小赛→大赛，每胜一档对手即一个小高潮。
- **高潮设计**：决赛/世界杯级舞台的终极胜利，完成封神。

## 5. 常见雷区/禁忌
- 项目常识错误被球迷抓包。
- 比赛描写流水账无紧张感。
- 主角无失败永远赢，失去逆袭味。
- 感情线喧宾夺主。
- 过度魔改真实球员/赛事引争议。
