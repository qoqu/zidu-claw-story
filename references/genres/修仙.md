# 修仙题材模板

> **核心卖点**：凡人逆天改命、一步步爬向长生顶点的阶层跃迁爽感，配合"我命由我不由天"的孤勇情绪。

## 1. 核心流派/细分
- **凡人流**：资质平庸的主角靠谨慎、算计与机缘慢慢爬，爽点在于"稳"和"反杀"，读者代入感最强。
- **无敌流/扮猪吃虎**：开局即强或很快变强，靠身份差制造打脸，节奏快、解压。
- **洪荒/神话流**：借用封神、西游等公共神话骨架，世界观宏大，适合写阵营博弈。
- **长生流/种田修仙**：低冲突、重日常与积累，靠时间跨度和心境描写黏住佛系读者。

## 2. 世界观/设定要素
- **境界体系**：炼气→筑基→金丹→元婴→化神……需明确每一大境界的能力天花板与寿命跨度。
- **资源链**：灵石、灵脉、丹药、功法、法宝构成硬通货，推动"夺宝—修炼—突破"循环。
- **宗门/王朝格局**：正邪对立、散修生存、上古秘境，提供冲突舞台。
- **天劫与因果**：突破需渡劫，杀戮有业力，给主角套上"变强必付出代价"的张力。

## 3. 经典爽点套路
- 废物退婚流：被悔婚→亮出底牌→对方跪求，经典打脸模板。
- 秘境夺宝：信息差捡漏，别人抢残羹他拿核心。
- 扮猪吃虎：被轻视后一招镇场，全场瞳孔地震。
- 师门护短：长辈出头替主角出气，满足"被偏爱"幻想。
- 跨阶反杀：以弱胜强，靠底牌/智谋翻盘。
- 收徒/建势力：从孤狼到一方巨擘的养成感。

## 4. 大纲节奏建议
- **开篇钩子**：用"退婚/测灵根垫底/被追杀"等羞辱性事件立人设，50章内完成第一次境界跃迁。
- **中期结构**：以"地图升级"推进（小宗门→中州→上界），每段换更大气的舞台，保持新鲜感。
- **高潮设计**：大型秘境争夺或正邪大战做阶段爆点，结尾留"上界之门"式悬念续命。

## 5. 常见雷区/禁忌
- 境界 numbering 混乱、战力崩坏，是读者弃书第一原因。
- 无脑杀伐、动辄屠城易踩暴力审核红线。
- 女主工具人化、后宫无脑争宠易被喷。
- 前期慢热无钩子，三章抓不住人直接沉。
- 抄设定不抄内核，世界观空有名词没有逻辑。
