# 直播文题材模板

> **核心卖点**：镜头前的真实与剧本交织，爽点是"用内容和人设从 0 粉到顶流"的流量逆袭，自带弹幕围观的沉浸狂欢。

## 1. 核心流派/细分
- **美食/生活直播**：做饭、赶海、下乡，治愈流量流。
- **户外/探险直播**：深山、探店、极限，刺激吸粉。
- **才艺/带货**：唱歌跳舞到直播卖货致富。
- **异能/系统直播**：主角能力被直播间围观，脑洞变现。

## 2. 世界观/设定要素
- **平台机制**：算法、打赏、热度榜，流量逻辑驱动剧情。
- **弹幕视角**：用观众评论做即时反馈与喜剧节奏。
- **人设经营**：真实感 vs 剧本的矛盾张力。
- **变现链条**：礼物、带货、商务，给成长数值。

## 3. 经典爽点套路
- 翻车反转：直播意外却爆红。
- 打脸黑粉：用实力让喷子闭嘴。
- 弹幕名场面：全员刷"主播牛"的狂欢。
- 助农带货：帮村民卖光滞销品。
- 流量反哺：主播用影响力做善事。

## 4. 大纲节奏建议
- **开篇钩子**：以"第一次开播就遇奇事/被嘲没人看"立起点。
- **中期结构**：每场直播一个话题爆点，粉丝数与事件难度同涨。
- **高潮设计**：大型直播事件（救援/破纪录）冲顶热搜，完成顶流跃迁。

## 5. 常见雷区/禁忌
- 平台机制写假失去真实感。
- 弹幕堆砌变成水字数。
- 直播内容流水账无看点。
- 带货变硬广惹烦。
- 过度渲染网红乱象失正向。
