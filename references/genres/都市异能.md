# 都市异能题材模板

> **核心卖点**：平凡城市里藏着超能力者与暗世界，爽点是"白天社畜晚上超人"的双重身份反差与低调装逼。

## 1. 核心流派/细分
- **隐世强者**：高手隐居都市被逼出手，扮猪吃虎。
- **异能组织**：主角入秘密机构执行任务，世界观向。
- **觉醒流**：普通青年意外获能闯异能圈。
- **都市修真/异能混合**：灵气复苏下的现代修炼。

## 2. 世界观/设定要素
- **表里世界**：日常都市+地下异能圈双层结构。
- **异能体系**：元素、精神、肉体等分类与等级。
- **组织/势力**：官方局、地下会、境外势力，博弈舞台。
- **能力代价**：透支、反噬，限制保张力。

## 3. 经典爽点套路
- 低调装逼：被嘲后随手露一手镇场。
- 异能破案：用能力解决官方难题。
- 守护日常：暗中护住普通人爱人。
- 组织晋升：从新人到王牌。
- 跨界大战：异能圈战争波及都市。

## 4. 大纲节奏建议
- **开篇钩子**：以"一次意外觉醒/被神秘人找上门"立双重身份。
- **中期结构**：小任务→组织战→大危机，异能规模升级。
- **高潮设计**：表里世界碰撞的终极战，主角以成长守住两边。

## 5. 常见雷区/禁忌
- 异能设定混乱战力崩。
- 都市感丢失变纯玄幻。
- 主角无敌无冲突平淡。
- 官方机构描写敏感引争议。
- 能力代价不设，随手秒天。
