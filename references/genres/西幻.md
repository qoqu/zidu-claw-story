# 西幻题材模板

> **核心卖点**：剑与魔法、巨龙与城堡的异世界浪漫，爽点是"凡人踏上英雄之旅"的史诗感与魔法体系的智性美感。

## 1. 核心流派/细分
- **骑士/冒险流**：组队下副本、屠龙救公主，经典英雄旅。
- **魔法学院流**：霍格沃茨式成长，校园+危机。
- **王权争霸**：领主/公主争王座，权谋+魔法。
- **龙裔/神裔**：血脉觉醒的宿命史诗。

## 2. 世界观/设定要素
- **魔法体系**：元素、咒语、魔网，规则清晰才有爽。
- **种族版图**：人、精灵、矮人、兽人，阵营与文化。
- **力量等级**：见习→大魔导→传奇，量化成长。
- **教会/王国**：权力与信仰结构，提供冲突。

## 3. 经典爽点套路
- 天赋觉醒：废柴测出 S 级属性。
- 魔法创新：组合咒语越级胜。
- 屠龙成名：斩获龙晶成英雄。
- 王座加冕：从流亡者到君主。
- 异族同盟：联合精灵破人类内战。

## 4. 大纲节奏建议
- **开篇钩子**：以"觉醒仪式垫底/家园被毁"立起点与动机。
- **中期结构**：学院/冒险→大陆事件→阵营战争，舞台持续放大。
- **高潮设计**：对抗远古 evil/加冕时刻，完成英雄宿命。

## 5. 常见雷区/禁忌
- 魔法规则混乱，战力崩。
- 过度模仿热门西幻 IP 被指融梗。
- 种族描写刻板化。
- 翻译腔过重失去网文节奏。
- 宗教设定处理敏感引争议。
