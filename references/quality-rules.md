| 禁止句式 | 替换方案 | 毒性 |
|---------|---------|------|
| "不是A，而是B" | 直接写B或用更自然的表达 | ★★★★★ |
| "，带着X"万能状语 | 拆短句或换具体动作 | ★★★★ |
| "声音不大，却带着X" | 直接写声音特征或动作 | ★★★★ |
| "他/她知道X都Y" | 用行为展示认知 | ★★★★ |
| "仿佛/如X般/似X一般" | 删掉或白描 | ★★★ |
| "感到X涌上心头" / "心中涌起X" | 用身体反应代替 | ★★★ |
| "他感到一股X涌上心头" | 用动作/细节代替 | ★★★ |
| "她的X宛如Y" | 删掉比喻或换新奇比喻 | ★★★ |
| "这不仅X，更Y" | 只说一个 | ★★ |
| "既X，又Y" | 只说一个 | ★★ |
| "X的同时，Y" | 分成两句 | ★★ |
| "这一刻，X" | 直接写X | ★★ |
| "面对X，他选择了Y" | 直接写他Y了 | ★★ |
| "尽管X，然而Y" | 直接写Y | ★★ |
| "终于明白X" / "这才意识到X" | 删或改动作 | ★★ |
| "原来，X" / "这就是X" | 删或改动作 | ★★ |
| "他不知道的是..." | 用具体钩子物件/事件收束 | ★★ |
| "不难看出/由此可见/综上所述" | 删掉（论文体，不入小说） | ★ |
| "于是乎/与此同时/从而/因而" | 口语化或直接删 | ★ |

## 段落规则

- 一段不超过 4 行
- 一段只承载一个动作/信息变化
- 单句超过 45 字拆短
- 段落超过 60 字按句号/动作转折拆开

## 对话规则

- 加入不完整句、打断、废话，口语化
- 不同角色说话方式要区分
- 对话标签（"他说""她道"）不超过 40%

## 心理描写规则

- **每次不超过 2 句**
- **优先用动作替代（能动手别动心）**
- 禁止以下心理直述模式（按AI味浓度排序）：
  - 一级阻断：他/她感到X、他/她感受到X、他/她觉得X、内心X、脑海浮现X
  - 二级阻断：他/她意识到X、他/她明白X、他/她清楚X（非对话中的认知直述）
  - 三级警告：心中X、心里X、在心底X、暗自/暗暗X、忍不住X、不由得X、不禁X
  - 四级警告：心想/心说/心道/心知/心念/心底X
- 心理直述密度门禁：轻度＜5处/千字 | 中度5-10处/千字 | 重度＞10处/千字
- 口诀：**情绪用动作展示，心理用身体反应替代**

| 心理直白 | 动作替代 |
|----------|----------|
| 他感到非常愤怒 | 他一拳砸在桌上 |
| 她心里很难过 | 她低头搅杯子，半天没喝 |
| 他觉得很温暖 | 他把围巾往上拽了拽 |
| 她感到无比失落 | 她把手机翻过去扣在桌上 |
| 他意识到自己错了 | 他张了张嘴，又闭上了 |
| 她在心底暗想 | 她盯着杯子出神 |

## 比喻规则

- 每千字不超过 1 个比喻
- 必须是新奇比喻（读者没见过的）

## 节奏规则

- 紧张处连续短句（3-8 字）
- 舒缓处允许长句
- 关键转折一句极短独立成段

## 留白规则

- 最强烈的感情不写
- 最重要的转折不解释
- 最美的画面不形容

## 禁止排比

- 禁止连续 3 句以上相同结构的排比

## 百分制评分规则

### 双层门禁机制

质量门禁采用双层机制：
1. **第一层：脚本检查**（客观项）— 禁用词/一致性/字数/跨章/声音/情绪/爽点
2. **第二层：LLM评分**（主观项）— 15维度百分制评分

章节必须同时通过两层才能继续：
- 第一层：无 blocker（退出码 ≠ 2）且无 warning（退出码 = 0 或 1）
- 第二层：评分 ≥ 90 分

### 退出码说明

| 退出码 | 含义 | 触发条件 |
|--------|------|----------|
| 0 | pass | 全部检查通过，评分 ≥ 90 |
| 1 | warn | 有警告但无阻断，评分 ≥ 90 |
| 2 | blocked | 有阻断项（必须修复） |
| 3 | score_fail | 规则检查通过但评分 < 90 |

### 15 个评分维度

| 维度 | 满分 | 评判重点 |
|------|------|---------|
| 开场吸引力 | 8 | 悬念钩子、冲突引入、亲和力开场 |
| 情感深度 | 8 | 内心变化、共鸣代入、情感层次 |
| 叙述结构 | 8 | 层次递进、结构紧凑、逻辑清晰 |
| 语言生动性 | 7 | 视觉语言、通俗易懂、多样化叙述 |
| 细节描写 | 7 | 画面感、具体生动、悬念保持 |
| 过渡连贯 | 6 | 自然过渡、节奏紧凑、思考空间 |
| 逻辑清晰 | 7 | 观点有力、深入剖析、立场鲜明 |
| 对比反差 | 6 | 主题突出、情感冲突、冲击力 |
| 情感张力 | 7 | 张力吸引、高潮引导、冲击力 |
| 人性化亲和 | 6 | 贴近生活、引发共鸣、故事性 |
| 视觉感官 | 7 | 视觉化、多感官、沉浸感 |
| 节奏流畅 | 7 | 紧凑不拖、节奏感、速度控制 |
| 逻辑连贯 | 6 | 情节连贯、时间线清晰、因果合理 |
| 细节具体化 | 5 | 具体例子、推动情节、悬念揭示 |
| 情感爆发力 | 5 | 积累自然、爆发有力、记忆点 |

### 题材模板

15 套题材模板位于 `score-templates/` 目录，通过 `--genre` 参数选择：
- default（默认/通用）、xuanhuan（玄幻）、xianxia（仙侠）、dushi（都市）
- xuanyi（悬疑）、yanqing（言情）、lishi（历史）、kehuan（科幻）
- moshi（末世）、chongsheng（重生）、chuanyue（穿越）、xitong（系统流）
- wuxianliu（无限流）、gongdou（宫斗/宅斗）、duanpian（短篇）

每套模板调整各维度权重分配，以适应不同题材的侧重需求。

### 评分不达标处理

1. 定位低分维度（score < 满分60%的维度）
2. 针对低分维度生成修复指令
3. 子 agent 执行定向修复（仅改低分维度涉及的内容）
4. 修复后重新评分
5. 最多 3 轮修复
6. 3 轮后仍 < 90 → 阻断，输出最终评分报告

## AI味七维免疫清单（写作前置约束）

> 写作前逐项阅读，写作时主动避开，不是写完再改。

### ① 语言规整流畅问题

AI 的句子太平稳工整，每句都"对"，但读多了乏味。

| 免疫规则 | 说人话版 |
|----------|----------|
| 避免连续3句以上相同句式结构 | 长短句交替，不能全是"主谓宾" |
| 每段至少有一句口语化表达 | 加入"行吧""算了""得了"等口语词 |
| 不同角色说话习惯要区分 | 口癖、节奏、用词偏好不同 |
| 适度加入停顿和语气词 | 嗯/啊/那个/就是/怎么说呢 |
| 不要所有句子都完整 | 可以断句、倒装、省略主语 |

**检查方法**：朗读一段感受节奏——如果所有句子读起来节奏一样，就是AI味。

### ② 逻辑平稳无个性问题

AI 的叙事逻辑四平八稳，缺乏强烈的个人立场。

| 免疫规则 | 说人话版 |
|----------|----------|
| 避免"一方面……另一方面……"结构 | 八股文结构，直接去掉 |
| 角色观点必须强烈 | 不能永远中立，要有偏向 |
| 情绪要有起伏 | 不能一直平稳叙述，加入情绪转折 |
| 每章至少有一个"非常规选择" | 角色做的事不能永远"合理" |
| 冲突中的双方观点不可调和 | 不要"双方都有道理"的和稀泥写法 |

### ③ 举例泛化问题

AI 举的例子总是泛泛而谈，缺少只有真实经历才有的细节。

| 免疫规则 | 说人话版 |
|----------|----------|
| 杜绝"比如一个人……""例如某天……" | 模糊开头直接删 |
| 描写加入具体细节 | 时间、地点、物件、人物表情、动作 |
| 场景加入五感细节 | 气味（消毒水/油烟/青草）、温度（闷热/凉飕飕）、触感（黏糊糊/扎手）、声音（冰箱嗡鸣/远处车声） |
| 角色互动加入小动作 | 抠指甲、抖腿、弹灰、转杯子、扯衣角 |

### ④ 立场模糊问题

AI 爱用模糊说法来"兜底"，避免承担责任。

| 免疫规则 | 说人话版 |
|----------|----------|
| 禁止"总体而言/在一定程度上/需结合实际情况" | 直接删，改成明确判断 |
| 角色判断必须有明确立场 | 不能"既……又……"两边都不得罪 |
| 冲突要有不可调和的矛盾 | 不是误会，是价值观/利益的根本冲突 |

### ⑤ 高频表达重复问题

AI 爱重复使用某些连接词和语气词。

| 免疫规则 | 说人话版 |
|----------|----------|
| "总之/因此/然而/不过"每千字≤2次 | 数一数，超了就删 |
| 不要连续段落以相同连接词开头 | 每段开头换花样 |
| 同一表达不重复使用3次以上 | 同一个词/句式在一章内出现3次就换 |


### ⑥ 事实精准问题

AI 常给出看起来合理但经不起核对的信息。

| 免疫规则 | 说人话版 |
|----------|----------|
| 设定信息（日期/数字/地名）前后一致 | 用追踪文件记录，不凭记忆写 |
| 避免"据说……""传说中……"作为设定替代 | 要么写实，要么不写 |
| 引用现实信息必须查证 | 法规/科学概念/历史事件不能编 |

### ⑦ 结构模板化问题

AI 每章的分段、节奏、过渡都太整齐。

| 免疫规则 | 说人话版 |
|----------|----------|
| 段落长短参差 | 不能每段4-5行均匀排列 |
| 避免"第一…第二…第三…"模板分段 | 叙事不是写报告 |
| 章节之间过渡有变化 | 不能每章都用同一种悬念收尾 |
| 可整段删掉不影响理解的内容就是废话 | 写完检查：删掉试试 |
